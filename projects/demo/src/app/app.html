Without ngx-call

openConfirm() {
  this.show = true;
}

onConfirm(value: boolean) {
  this.result = value;
  this.show = false;
}

With ngx-call

async openConfirm() {
  const ok = await Confirm.call({
    message: 'Continue?'
  });

  if (ok) {
    // Continue the flow.
  }
}

Declared once. Called from anywhere.

The host lives in your Angular tree. The call happens wherever your flow needs an answer.

1

Wrap a standalone component with inputs and a root context.

2

Call

Use Confirm.call() and receive a promise immediately.

4

Resolve

The component ends the call with the value the calling flow awaits.

State vs. Await

Awaitable UI replaces scattered open flags with one straight-line flow.

  • More code
  • Scattered updates
  • Harder to test
  • Manual cleanup
let open = signal(false);
let answer = signal<boolean | null>(null);

open.set(true);

effect(() => {
  if (answer() !== null) {
    // Continue...
  }
});
const answer = await Confirm.call({
  message: 'Proceed?'
});

if (answer) {
  // Continue...
}
  • Less code
  • Straight-line flow
  • Easier to test
  • No manual cleanup
